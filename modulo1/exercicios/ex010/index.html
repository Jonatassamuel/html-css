Listas:

Listas Ordenadas (Ordered):

São aquelas em que a ordem do elementos são importantes, para o significado da mesma.

  1. teste 1
  2. teste 2
  3. teste 3
  4. teste 4
  5. teste 5

estrutura da tag:

<ol>

<li>...</li>
<li>...</li>
<li>...</li>
<li>...</li>
<li>...</li>

</ol>

A tag <lo> é o delimitador da lista e a tag <li> é o conteúdo da lista

Algumas configurações:

A tag <ol>, possui um parâmetro type, onde configuramos o tipo de marcador da lista atual.


Layout da conf: <ol type="A" start = "5"

Tipos de marcadores:
  • type = 1- Valor padrão. Cria listas númericas. Ex: 1,2,3,4,...
  • type = A - Cria listas alfabéticas em maiúsculas; Ex: A,B,C,D,...
  • type = a - Cria listas alfabéticas em minúsculas. Ex:a,b,c,d,...
  • type = I - Cria listas com algarismos romanos em maiúsculas. Ex: I,II,III,IV,...
  • type = i - Cria listas com algarismos romanos em minúsculas. Ex: i,ii,iii,iv,...


Listas Não Ordenadas (Unordered):

São aqueles em que a ordem não altera o resultado/significado da lista.


Estrutura da Unordered:


<lu>

<li>...<li>
<li>...<li>
<li>...<li>
<li>...<li>
<li>...<li>

<lu>

A tag <lu> é o delimitador da lista e a tag <li> e o conteúdo

Sub-listas:

É a possibilidade de criar listas dentro de listas, Ex:


  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
  5. Item 5
