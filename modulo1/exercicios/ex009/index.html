Teste de Tags:

Aqui estão algumas tags importantes para formatação de textos:

TAGS:

Tag negrito; <strong>...</strong>

Apartir da quebra de linha o texto ficará na formatação da tag:


Lorem ipsum dolor sit amet, consectetur adipisicing elit. Labore saepe ipsam in quibusdam optio facere molestiae doloribus laborum sunt error consequatur eaque, nostrum consectetur natus sed placeat neque sint quam.


Tag itálico; <em>...</em>

Apartir da quebra de linha o texto ficará na formatação da tag:


Lorem ipsum dolor sit amet consectetur adipisicing elit. Cumque itaque dolores esse, tenetur perferendis optio veritatis saepe nisi dolore commodi nulla, provident inventore quidem magni. Laudantium molestias ducimus reprehenderit ea.>


Tag marca-texto; <mark>...</mark>

Sabe o marca texto que você usa no caderno? então essa tag é a mesma coisa. apartir da quebra de linha vamos destacar 5 palavras:


Lorem ipsum dolor sit amet, consectetur adipisicing elit. Ex ipsam odio mollitia maxime, quaerat, maiores culpa a laboriosam neque sint repellat id nostrum, quos praesentium necessitatibus rem illo animi aperiam.


Tag texto deletado; <del>...<del>

Para ter uma determinada parte do seu texto na formatação demonstrada abaixo use a tag que está acima 👆


ipsum dolor sit amet consectetur adipisicing elit. Tempora corporis aliquid ea, reprehenderit beatae eligendi amet ex saepe minima quam, et mollitia modi natus dignissimos autem laudantium in, facere consectetur!


Tag texto inserido; <ins>...</ins>

Para da destaque a uma parte importante do texto como na demonstração abaixo use a tag acima 👆


ipsum dolor sit amet consectetur adipisicing elit. Soluta ad ut doloremque suscipit reprehenderit, ipsam voluptas animi sed tempora saepe sequi temporibus distinctio illo. Eius repudiandae ut vitae omnis? Veritatis!


Tag para exibir um código; <pre>...</pre>

Vamos exibir um código em Python de um leitor de números pares ou ímpares


def eh_par(numero):
        """
        Verifica se o número é par.
      
        Argumentos:
          numero: O número a ser verificado.
      
        Retorno:
          True se o número for par, False caso contrário.
        """
        return numero % 2 == 0
      
      def main():
        """
        Le um número do usuário e informa se ele é par ou ímpar.
        """
        numero = int(input("Digite um número: "))
      
        if eh_par(numero):
          print(f"O número {numero} é par.")
        else:
          print(f"O número {numero} é ímpar.")
      
      if __name__ == "__main__":
        main()

❗Observação importante: para exibir um código, eu posso usar a tag: <pre>...</pre> e <code>...</code>

Textos sobrescritos e subscritos

Para colocarmos termos subscritos usamos a tag: <sub>...</sub>
EX: a 1, 1 a


E termos sobrescritos usamos a tag: <sup>...</sup>
EX: 1 a a 1


Tag de citação: <q>...</q>

Como diria um sabio professor: O computador é um burro muito rápido

Citação completa: <blackquote>...</blackquote>

Eu acredito no amor. Eu creio que pode existir alguma chance e eu sempre acho que pode ser diferente. Talvez essa seja a explicação de todas decepções.
