Links:

O link ou ligação é a essência fundamental do hipertexto. Elas nos permite ligar documentos entre si e permitir a navegação entre essas páginas.


Âncoras:

Os hyperlink são um dos conceitos mais antigos da história da linguagem HTML. Eles permitem que você ligue um ponto a outro na World Wide Web. Toda vez que você está acessando um site e clica em um local par ir para outra página, outro site ou até para baixar um arquivo, Nós estamos interagindo com um hyperlink


Para criarmos um hyperlink, devemos criar uma âncoras através da tag <a>. Essas tag possui um atributo muito importante que é o href;

< href="https://https://github.com/gustavoguanabara/html-css/tree/master/aulas-pdf">Perfil do GitHub >

Destino do link:

Como visto anteriormente a tag <a> tem o atributo href que possui alguns valores sendo eles;

  • _blank vai abrir o link em uma nova janela em branco
  • _self vai abrir o link na janela ou frama atual (padrão)

EX:

<a href="pagina2.html" target="_self">Continuar navegando no site </a>

<a href=""https://gustavoguanabara.github.io target="_blank">Abrir perfil do GitHub em nova janela</a>

Natureza do link:

Podemos indicar a natureza de um link usando o atributo rel. Esse atributo aceita alguns valores, sendo eles:

  • next indica que o link é para a próxima parte do documento atual
  • prev indica que o link é para a parte anterior do documento atual
  • author indica que é um link para o site do autor do artigo atual
  • external indica que é um link para outro site que não faz parte do site


EX:

href="pagina2.html" targe="_self" rel=""next
